# Icon & Logo Guidelines

The `icon.png` and `logo.png` files should be replaced with actual images:

## icon.png
- **Size**: 256x256 pixels
- **Format**: PNG with transparency
- **Design**: Simple, recognizable icon representing AI/LLM
- **Suggested**: Brain, neural network, or chip icon
- **Colors**: Blue/purple gradient or material design colors

## logo.png  
- **Size**: 512x256 pixels (or similar 2:1 ratio)
- **Format**: PNG with transparency
- **Design**: Horizontal banner with text "Local LLM"
- **Suggested**: Include icon + text
- **Colors**: Match icon colors

## Quick Creation Options

1. **Use Free Tools:**
   - [Canva](https://www.canva.com/) - Easy online design
   - [Figma](https://www.figma.com/) - Professional design tool
   - [GIMP](https://www.gimp.org/) - Free Photoshop alternative

2. **Use AI Image Generators:**
   - DALL-E, Midjourney, or Stable Diffusion
   - Prompt: "minimalist tech icon for AI server, blue gradient, 256x256"

3. **Use Existing Icons:**
   - [Material Icons](https://fonts.google.com/icons)
   - [Font Awesome](https://fontawesome.com/)
   - Search for: brain, cpu, chip, network

## Note

The add-on will work without custom icons - Home Assistant will use defaults. However, custom icons improve the professional appearance of your add-on.
